 Ultra Premium Age Difference Calculator
A production-level, high-performance web application designed with a focus on modern aesthetics and precise calculations. This project demonstrates advanced CSS techniques, smooth animations, and robust vanilla JavaScript logic.
✨ Features
Ultra-Premium UI: Glassmorphism design with multi-layer gradient blending and neon glow effects.
Precision Logic: Accurate calculation of age difference in Years, Months, and Days, accounting for leap years and varying month lengths.
Dynamic Animations:
Smooth fade-in entrance.
Interactive result counter animations.
Hover-triggered neon glow micro-interactions.
Subtle floating background motion.
Fully Responsive: Optimized for Mobile, Tablet, and Desktop displays.
Robust Validation: Real-time error handling for invalid date selections or logical inconsistencies (e.g., end date before start date).
🚀 Tech Stack
HTML5: Semantic structure for accessibility and SEO.
CSS3: Advanced styling including Flexbox, CSS Variables, and Keyframe animations.
JavaScript (ES6+): Pure vanilla JS for high performance without external dependencies.
Typography: Integrated Google Fonts (Inter & Outfit) for a professional look.

🛠️ Installation & Usage
Clone or Download: Extract the provided ZIP file to your local machine.
Open: Simply double-click index.html to open the application in any modern web browser.
Calculate: Select your start and end dates from the dropdowns and click CALCULATE to see the result with smooth animations.
